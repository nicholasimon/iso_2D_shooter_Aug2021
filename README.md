# iso_2D_shooter_Aug2021
A fake 3D (isometric) terrain arcade shooter game idea with 2D sprites
<br />
Arrow Keys - Move Ship<br />
Space - Fire<br />
Keypad * - Enables Map Move - Then use arrow keys to move the map<br />
Keypad +/- - Zoom controls<br />
End - Close Window<br />
