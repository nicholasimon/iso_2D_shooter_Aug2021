# iso_2D_shooter_Aug2021
A fake 3D (isometric) terrain arcade shooter game idea with 2D sprites

Arrow Keys - Move Ship
Space - Fire
Keypad * - Enables Map Move - Then use arrow keys to move the map
Keypad +/- - Zoom controls
End - Close Window
